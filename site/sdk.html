SDK Examples

Integrate Agent Identity in minutes.

Examples in Python, TypeScript, and curl.

Python

import requests

resp = requests.post(
    "https://identity.agennext.ai/v1/agents",
    json={"name": "Research Agent"}
)
print(resp.json())

TypeScript

const resp = await fetch('/v1/agents', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({ name: 'Research Agent' })
});

curl

curl -X POST https://identity.agennext.ai/v1/agents \
  -H 'Content-Type: application/json' \
  -d '{"name":"Research Agent"}'
